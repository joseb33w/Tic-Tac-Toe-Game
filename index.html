Classic Board Game

Tic Tac Toe

Take turns placing X and O. First to get three in a row wins!

Game Board

Tap any empty cell to make your move.

Turn: X

Scoreboard

Player X Wins 0
Player O Wins 0
Draws 0
Status X goes first. Good luck!

How to Play

  1. X always goes first
  2. Tap an empty cell to place your mark
  3. Get 3 in a row (horizontal, vertical, or diagonal) to win
  4. If all 9 cells fill up with no winner, it's a draw
